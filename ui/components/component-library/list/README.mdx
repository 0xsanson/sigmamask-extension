# List

List is an unstyled wrapping component for list items

## Props

<Controls of={ListStories.DefaultStory} />

### As

Using the box prop `as` on `List` will typically be set to `ul` or `ol`

Defaults to `ul`

<Canvas of={ListStories.As} />

```jsx
import { List } from '../../component-library';

<List>
  <li>ul item 1</li>
  <li>ul item 2</li>
</List>
<List as="ol">
  <li>ol item 1</li>
  <li>ol item 2</li>
</List>
```

### Children

The `children` of the `List` can be text or a react node

<Canvas of={ListStories.Children} />

```jsx
import { List } from '../../component-library';

<List>List children</List>;
```
