# ListItemSelect

List Item represents an item in a list where items are either read-only or clickable but not selectable

### Children

The `children` of the `ListItemSelect` can be text or a react node

<Canvas of={ListItemSelectStories.Children} />

```jsx
import { ListItemSelect } from '../../component-library';

<ListItemSelect>ListItemSelect children</ListItemSelect>;
```

### isDisabled

The `isDisabled` boolean prop of the `ListItemSelect` determines whether the item is disabled or not

<Canvas of={ListItemSelectStories.IsDisabled} />

```jsx
import { ListItemSelect } from '../../component-library';

<ListItemSelect>List item</ListItemSelect>;
```

### isSelected

The `isSelected` boolean prop of the `ListItemSelect` determines whether the item is selected

<Canvas of={ListItemSelectStories.IsSelected} />

```jsx
import { ListItemSelect } from '../../component-library';

<ListItemSelect isSelected>List item</ListItemSelect>;
```
